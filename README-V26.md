# Orchestra Lartife V26 — KR / EN / JP / CN

Complete four-language static website.

Language naming:
- Korean: original filename, e.g. index.html
- English: -en.html
- Japanese: -jp.html
- Simplified Chinese: -cn.html

The language switch on every page opens the corresponding page in the selected language.
The mobile hamburger menu also includes KR / EN / JP / CN shortcuts.

The Korean conductor biography retains:
오케스트라 라티프를 통해 음악과 사람을 연결하는 다양한 공연과 예술 프로젝트를 기획하며, 국내외 아티스트와의 협업을 바탕으로 새로운 클래식 음악의 가능성을 꾸준히 확장해가고 있다.
